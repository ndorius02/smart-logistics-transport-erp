# Functional Analysis

## Project Name

Smart Logistics & Transport ERP

## Project Overview

Smart Logistics & Transport ERP is a full-stack enterprise application designed to simulate real logistics, transport, cargo and supply chain operations.

The application centralizes business data related to customers, suppliers, carriers, products, warehouses, inventory, purchase orders, cargo, shipments, deliveries, drivers, vehicles and routes.

## Business Context

Logistics and transport companies need reliable software systems to manage daily operations, track goods, coordinate transport, monitor stock levels and ensure accurate data across the supply chain.

This project is inspired by real business processes found in logistics companies, transport operators, cargo operations, import/export businesses and ERP environments.

## Project Objectives

The main objectives of the application are to:

- Manage business partners such as customers, suppliers and carriers
- Manage products, warehouses and inventory
- Manage purchase orders and purchase order lines
- Track cargo, shipments and deliveries
- Manage drivers, vehicles and routes
- Monitor operational performance through dashboards and KPIs
- Provide a structured ERP-style system using Angular, Spring Boot and PostgreSQL

## Main Users

The application is designed for the following user profiles:

- Administrator
- Logistics Manager
- Supply Chain Officer
- Transport Coordinator
- Warehouse Officer
- Back Office User
- Application Support User

## ERP Modules

### Administration

This module manages users, roles, authentication and authorization.

Main entities:

- User
- Role

### Business Partners

This module manages external partners involved in the supply chain.

Main entities:

- Customer
- Supplier
- Carrier

### Warehouse Management

This module manages warehouses, products and stock levels.

Main entities:

- Product
- Warehouse
- Inventory

### Procurement

This module manages supplier orders.

Main entities:

- PurchaseOrder
- PurchaseOrderLine

### Cargo & Shipment Management

This module manages cargo, shipments and delivery tracking.

Main entities:

- Cargo
- Shipment
- Delivery
- DeliveryStatus

### Transport Management

This module manages drivers, vehicles and routes.

Main entities:

- Driver
- Vehicle
- Route

### Reporting & Dashboard

This module provides key operational indicators.

Examples of KPIs:

- Total shipments
- Deliveries in progress
- Delayed deliveries
- Available vehicles
- Warehouse occupancy
- Open purchase orders
